### VS Code 打开方式

1. 下载安装 [VS Code](https://code.visualstudio.com/)（免费）
2. **打开文件夹**：`文件` → `打开文件夹` → 选择解压后的 `code_doc` 文件夹
3. **查看目录**：点击左侧任意 `.md` 文件 → 按 `Ctrl+Shift+P` → 输入 `Markdown: Open Preview` 即可预览
4. **大纲导航**：右侧「大纲」面板（Outline）可快速跳转到任意题目
5. **全局搜索**：`Ctrl+Shift+F` 可在所有文件中搜索关键词（如"互斥锁"、"DMA"）

> 💡 **小技巧**：安装 VS Code 插件 **Markdown All in One** 可获得更好的阅读体验（目录生成、快捷键等）

### 导出为 PDF

- **VS Code**：安装插件 **Markdown PDF** → 右键 `.md` 文件 → `Markdown PDF: Export (pdf)`
- **Typora**：`文件` → `导出` → `PDF`
- **在线工具**：将 `.md` 内容粘贴到 [dillinger.io](https://dillinger.io) → 导出 PDF
